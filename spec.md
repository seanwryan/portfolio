# Portfolio Project Specification

## Goal
Build a modern, high-performance personal portfolio website using Next.js 14+, Tailwind CSS, and Framer Motion for animations. The design should be "Research Lab" aesthetic: minimalist, clean typography, plenty of whitespace.

## Tech Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion
- Icons: Lucide React
- Deployment: Vercel

## Core Pages
1. **Home:** Hero section with a bold one-liner.
2. **Projects:** A grid layout of project cards. Each card has a thumbnail, title, tech stack tags, and a "View Case Study" button.
3. **About:** A "Chat with my Resume" section (UI only for now).
4. **Blog/Log:** A simple list of recent AI experiments.

## Design Requirements
- Font: Inter or Geist Mono (Google Fonts).
- Background: Off-white (#F5F5F7) with dark text (#111).
- Mobile responsive.
